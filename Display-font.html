**********Difference Between C language and C++ language***********
C language
1.C was developed by Denis Richie.
2.C is subset of C++.
3.C is proceduceral programming language.
4.C doesnot support reference variables.
5.C doesnot have NAMESPACE feature.
C++ language
1.C++ was developed by Bjarne Stroustrup.
2.C++ is superset of C.
3.C++ is object orientd programming language.
4.C++ support reference variables.
5.C++ have NAMESPACE feature.
